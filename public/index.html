¿Como cambiar el estilo de cualquier texto?

El texto al que se requiera cambiar su estilo debe ser rodeado por alguno de los caracteres especiales que estan definidos en este generador.
Es importante entender que el cambio solo se verá reflejado en el OVA.


Caracteres especiales:

Para resaltar un texto: ***

Para aplicar el estilo italica a un texto: ---


Ejemplos:

Para resaltar un texto aumentando el ancho de las letras: "En este texto ***quiero resaltar esta parte*** pero no quiero afectar al resto del texto"

Para aplicar el estilo italica a un texto: "En este otro texto ---quiero aplicar el estilo italica--- pero no quiero afectar al resto del texto"
